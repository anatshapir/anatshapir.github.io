מדריך למתחילים: פיתוח אתר ב-ASP.NET

חלק 1: הכרת הסביבה ויצירת דף ראשון

למי מיועד המדריך? המדריך מיועד לתלמידי מדעי המחשב שאין להם ניסיון קודם בפיתוח אתרים. נתחיל מאפס ונלמד צעד אחר צעד.

צעד 1: התקנת Visual Studio

Visual Studio היא סביבת הפיתוח (IDE) שבה נשתמש לבניית האתר שלנו. היא מספקת לנו את כל הכלים הנדרשים לכתיבה, עריכה, ובדיקה של הקוד שלנו.

איך מתקינים?

  1. הורידו את Visual Studio Community (גרסה חינמית) מהאתר הרשמי של Microsoft
  2. הפעילו את תוכנת ההתקנה
  3. בחרו ב-"ASP.NET and web development" בין רכיבי ההתקנה
  4. המשיכו בהתקנה עד לסיומה

חשוב: וודאו שבחרתם ב-"ASP.NET and web development" אחרת לא יהיו לכם הכלים הנדרשים לפיתוח אתרים.

צעד 2: הכרת הממשק של Visual Studio

כאשר פותחים את Visual Studio בפעם הראשונה, התוכנה עשויה להיראות מורכבת. בואו נכיר את החלקים העיקריים:

חלקים חשובים בממשק:

  • תפריט ראשי - נמצא בחלק העליון ומכיל אפשרויות שונות כמו File, Edit, View, וכו'.
  • Solution Explorer - בצד ימין, מציג את כל הקבצים והתיקיות בפרויקט שלכם.
  • Toolbox - בצד שמאל, מכיל כלים ופקדים שניתן לגרור לדפים שלכם.
  • Properties - בדרך כלל בצד ימין התחתון, מציג ומאפשר לערוך מאפיינים של האלמנט הנבחר.
  • שטח עבודה מרכזי - האזור הגדול במרכז שבו תערכו את הקוד שלכם.

טיפים לשימוש ב-Visual Studio:

  • Ctrl+S - שומר את הקובץ הנוכחי.
  • Ctrl+Shift+S - שומר את כל הקבצים.
  • F5 - מריץ את הפרויקט עם דיבוג (אפשר לעצור את התוכנית באמצע הריצה).
  • Ctrl+F5 - מריץ את הפרויקט ללא דיבוג (מהיר יותר).
  • Ctrl+Space - מציג השלמה אוטומטית של קוד.

צעד 3: יצירת פרויקט ASP.NET חדש

עכשיו נלמד איך ליצור פרויקט חדש:

  1. פתחו את Visual Studio
  2. לחצו על "Create a new project" בחלון ההתחלתי
  3. בחלון שנפתח, חפשו וסננו לפי: "ASP.NET Web Application (.NET Framework)"
  4. לחצו על "Next"
  5. הזינו שם לפרויקט שלכם (לדוגמה: "MyFirstWebsite")
  6. ודאו שמיקום הפרויקט (Location) הוא תיקייה שאתם זוכרים ויכולים לגשת אליה
  7. לחצו על "Create"
  8. בחלון הבא, בחרו בתבנית "Empty" (ריקה) וסמנו את התיבה "Web Forms"
  9. לחצו על "Create"

חשוב לשים לב לבחירה ב"Empty" עם סימון "Web Forms". זה ייצור לנו פרויקט ASP.NET בסיסי שנוכל להרחיב בקלות.

אחרי יצירת הפרויקט, תוכלו לראות בחלון Solution Explorer את המבנה הבסיסי של הפרויקט:

  • Properties - הגדרות בסיסיות של הפרויקט
  • References - הפניות לספריות שהפרויקט משתמש בהן
  • Web.config - קובץ תצורה המכיל הגדרות שונות של האתר

צעד 4: הוספת תיקיות בסיסיות

לפני שנתחיל לבנות את האתר, כדאי להוסיף תיקיות שיעזרו לנו לארגן את הפרויקט בצורה נכונה:

  1. לחצו קליק ימני על שם הפרויקט (בחלון Solution Explorer)
  2. בחרו Add > New Folder
  3. צרו את התיקיות הבאות:
    • CSS - לקבצי עיצוב
    • Images - לתמונות
    • Scripts - לקבצי JavaScript

וודאו גם שיש תיקייה בשם App_Data שתשמש אותנו לאחסון נתונים:

  1. אם אינה קיימת, לחצו קליק ימני על שם הפרויקט
  2. בחרו Add > Add ASP.NET Folder > App_Data

צעד 5: יצירת דף HTML ראשון

ASP.NET Web Forms משתמש בקבצי .aspx, שהם סוג של דפי HTML עם יכולות נוספות. עכשיו נייצר את הדף הראשון שלנו:

  1. לחצו קליק ימני על שם הפרויקט
  2. בחרו Add > New Item
  3. בחרו Web Form ותנו לו את השם "Default.aspx" (זהו שם חשוב מכיוון שדפדפנים מחפשים דף בשם זה כברירת מחדל)
  4. לחצו על "Add"

עכשיו ייפתח חלון העריכה עם הקוד ההתחלתי של הדף:

            <%@ Page Language="C#" AutoEventWireup="true" CodeBehind="Default.aspx.cs" Inherits="MyFirstWebsite.Default" %>

            <!DOCTYPE html>

            <html xmlns="http://www.w3.org/1999/xhtml">
            <head runat="server">
            <title></title>
            </head>
            <body>
            <form id="form1" runat="server">
            <div>
            </div>
            </form>
            </body>
            </html>

בואו נבין את הקוד הזה:

  • <%@ Page ... %> - זוהי הוראה מיוחדת לשרת שאומרת שזהו דף ASP.NET. היא מקשרת בין דף ה-HTML (default.aspx) ובין קובץ הקוד מאחורי הקלעים (default.aspx.cs) שמכיל קוד C#.
  • <!DOCTYPE html> - מגדיר שזהו דף HTML5.
  • <html> - אלמנט השורש שמכיל את כל דף ה-HTML.
  • <head> - מכיל מידע על הדף (כמו כותרת, קישורים לקבצי CSS, וכו').
  • <title> - הכותרת שתופיע בלשונית בדפדפן.
  • <body> - מכיל את כל מה שיוצג בדפדפן.
  • <form runat="server"> - זהו טופס ASP.NET שמאפשר לנו להשתמש בפקדי ASP.NET. ה-runat="server" אומר שהטופס ינוהל בצד השרת.
  • <div> - אלמנט בסיסי שמשמש לקיבוץ של אלמנטים אחרים.

צעד 6: עריכת דף HTML בסיסי

עכשיו נערוך את הדף שלנו כדי שיכיל תוכן בסיסי. נוסיף כותרת, פסקה וכמה אלמנטים בסיסיים נוספים:

            <%@ Page Language="C#" AutoEventWireup="true" CodeBehind="Default.aspx.cs" Inherits="MyFirstWebsite.Default" %>

            <!DOCTYPE html>

            <html xmlns="http://www.w3.org/1999/xhtml">
            <head runat="server">
            <title>האתר הראשון שלי</title>
            <meta charset="utf-8" />
            </head>
            <body>
            <form id="form1" runat="server">
            <div>
            <h1>ברוכים הבאים לאתר שלי!</h1>
            <p>זהו האתר הראשון שלי ב-ASP.NET</p>

            <h2>נושאים באתר:</h2>
            <ul>
            <li>נושא ראשון</li>
            <li>נושא שני</li>
            <li>נושא שלישי</li>
            </ul>

            <p>לחצו <a href="#">כאן</a> לקריאה נוספת.</p>
            </div>
            </form>
            </body>
            </html>

השינויים שביצענו:

  • הוספנו כותרת בתג <title>
  • הוספנו <meta charset="utf-8" /> כדי שהדפדפן יוכל להציג עברית כראוי
  • הוספנו כותרת ראשית <h1>
  • הוספנו פסקת טקסט <p>
  • הוספנו כותרת משנית <h2>
  • הוספנו רשימה לא ממוספרת <ul> עם פריטים <li>
  • הוספנו קישור בתג <a>

הרצת האתר:

כדי לראות את הדף שלנו בפעולה:

  1. שמרו את הקובץ (Ctrl+S)
  2. לחצו על F5 או על הכפתור ▶️ (Start) בסרגל הכלים
  3. הדפדפן ייפתח ויציג את האתר שלכם

שימו לב: אם הטקסט בעברית לא מוצג כראוי, וודאו שהוספתם את השורה <meta charset="utf-8" /> בתוך תג ה-<head>.

צעד 7: למידה על HTML - תגיות בסיסיות

כעת נלמד על התגיות הבסיסיות ב-HTML שכדאי להכיר:

תגית תיאור דוגמה
<h1> עד <h6> כותרות בגדלים שונים (h1 הגדולה ביותר, h6 הקטנה ביותר) <h1>כותרת ראשית</h1>
<p> פסקה <p>זוהי פסקת טקסט.</p>

<img> תמונה <img src="Images/pic.jpg" alt="תיאור התמונה" />
<ul> ו-<li> רשימה לא ממוספרת ופריטי רשימה <ul>
  <li>פריט 1</li>
  <li>פריט 2</li>
</ul>
<ol> ו-<li> רשימה ממוספרת ופריטי רשימה <ol>
  <li>פריט 1</li>
  <li>פריט 2</li>
</ol>
<div> מיכל כללי לקיבוץ אלמנטים (בלוק) <div>תוכן כלשהו</div>
<span> מיכל לטקסט (שורה) <span>טקסט מודגש</span>
<br> שבירת שורה שורה ראשונה<br>שורה שנייה
<hr> קו אופקי טקסט<hr>טקסט נוסף

צעד 8: למידה על טבלאות בHTML

טבלאות הן דרך נוחה להציג מידע מסודר בשורות ועמודות. בואו נלמד איך ליצור טבלה בסיסית:

            <table>
            <caption>כותרת הטבלה</caption>
            <thead>
            <tr>
            <th>כותרת עמודה 1</th>
            <th>כותרת עמודה 2</th>
            <th>כותרת עמודה 3</th>
            </tr>
            </thead>
            <tbody>
            <tr>
            <td>נתון 1,1</td>
            <td>נתון 1,2</td>
            <td>נתון 1,3</td>
            </tr>
            <tr>
            <td>נתון 2,1</td>
            <td>נתון 2,2</td>
            <td>נתון 2,3</td>
            </tr>
            </tbody>
            </table>

הסבר על תגיות הטבלה:

  • <table> - מגדיר טבלה
  • <caption> - כותרת הטבלה (אופציונלי)
  • <thead> - מכיל את הכותרות של העמודות
  • <tbody> - מכיל את הנתונים העיקריים בטבלה
  • <tr> - מגדיר שורה בטבלה (Table Row)
  • <th> - מגדיר תא כותרת (Table Header), בדרך כלל מודגש ומיושר למרכז
  • <td> - מגדיר תא נתונים רגיל (Table Data)

מיזוג תאים בטבלה:

אפשר למזג תאים לרוחב או לגובה באמצעות המאפיינים colspan ו-rowspan:

            <table>
            <tr>
            <td colspan="2">תא שמתפרס על פני 2 עמודות</td>
            </tr>
            <tr>
            <td rowspan="2">תא שמתפרס על פני 2 שורות</td>
            <td>תא רגיל</td>
            </tr>
            <tr>
            <td>תא רגיל נוסף</td>
            </tr>
            </table>

בואו נוסיף טבלה לדף שלנו:

נוסיף טבלה פשוטה לדף שיצרנו:

            <h2>מערכת שעות:</h2>
            <table>
            <tr>
            <th>יום</th>
            <th>8:00-10:00</th>
            <th>10:00-12:00</th>
            <th>12:00-14:00</th>
            </tr>
            <tr>
            <td>ראשון</td>
            <td>מתמטיקה</td>
            <td>אנגלית</td>
            <td>מדעי המחשב</td>
            </tr>
            <tr>
            <td>שני</td>
            <td>היסטוריה</td>
            <td colspan="2">פרויקט מיוחד</td>
            </tr>
            <tr>
            <td>שלישי</td>
            <td>ספרות</td>
            <td>פיזיקה</td>
            <td>כימיה</td>
            </tr>
            </table>

הוסיפו את הקוד הזה לדף שלכם והריצו אותו כדי לראות את התוצאה. שימו לב שבשלב זה הטבלה תיראה בסיסית מאוד, ללא עיצוב.

צעד 9: למידה על טפסים בHTML

טפסים (forms) משמשים לאיסוף מידע מהמשתמשים. ב-ASP.NET, אנחנו כבר נמצאים בתוך טופס מובנה (<form runat="server">), אבל חשוב להבין את המבנה הבסיסי של טפסים ואיך משתמשים בהם:

רכיבי טופס בסיסיים:

סוג שדה תגית HTML תיאור דוגמה
שדה טקסט <input type="text"> שדה להזנת טקסט בשורה אחת <input type="text" name="username" />
סיסמה <input type="password"> שדה להזנת סיסמה (הטקסט מוסתר) <input type="password" name="password" />
תיבת סימון <input type="checkbox"> תיבה שאפשר לסמן או לא <input type="checkbox" name="agree" /> אני מסכים
כפתור רדיו <input type="radio"> כפתור בחירה (אפשר לבחור אחד מקבוצה) <input type="radio" name="gender" value="male" /> זכר
<input type="radio" name="gender" value="female" /> נקבה
תיבת בחירה <select> ו-<option> רשימה נפתחת לבחירה <select name="city">
  <option value="tlv">תל אביב</option>
  <option value="jer">ירושלים</option>
</select>
אזור טקסט <textarea> שדה להזנת טקסט במספר שורות <textarea name="comments" rows="4" cols="50"></textarea>
כפתור שליחה <input type="submit"> כפתור לשליחת הטופס <input type="submit" value="שלח" />
כפתור איפוס <input type="reset"> כפתור לאיפוס הטופס <input type="reset" value="נקה" />

שימו לב: המאפיין name הוא חשוב מאוד בשדות טופס! הוא משמש כמזהה של השדה כאשר הטופס נשלח.

דוגמה לטופס HTML בסיסי:

                        <form action="process.aspx" method="post">
                        <div>
                        <label for="fullname">שם מלא:</label>
                        <input type="text" id="fullname" name="fullname" />
                        </div>

                        <div>
                        <label for="email">אימייל:</label>
                        <input type="email" id="email" name="email" />
                        </div>

                        <div>
                        <label for="age">גיל:</label>
                        <input type="number" id="age" name="age" min="1" max="120" />
                        </div>

                        <div>
                        <label>מגדר:</label>
                        <input type="radio" id="male" name="gender" value="male" />
                        <label for="male">זכר</label>
                        <input type="radio" id="female" name="gender" value="female" />
                        <label for="female">נקבה</label>
                        </div>

                        <div>
                        <label for="city">עיר:</label>
                        <select id="city" name="city">
                        <option value="">בחר עיר</option>
                        <option value="tlv">תל אביב</option>
                        <option value="jer">ירושלים</option>
                        <option value="hfa">חיפה</option>
                        </select>
                        </div>

                        <div>
                        <label for="comments">הערות:</label>
                        <textarea id="comments" name="comments" rows="4" cols="30"></textarea>
                        </div>

                        <div>
                        <input type="checkbox" id="agree" name="agree" value="yes" />
                        <label for="agree">אני מסכים לתנאי השימוש</label>
                        </div>

                        <div>
                        <input type="submit" value="שלח" />
                        <input type="reset" value="נקה" />
                        </div>
                        </form>

הסבר על מאפייני טפסים חשובים:

  • action - כתובת הדף שאליו יישלחו הנתונים כשהטופס מוגש
  • method - שיטת השליחה (get או post). בדרך כלל משתמשים ב-post לטפסים
  • id - מזהה ייחודי לאלמנט, משמש בעיקר ל-JavaScript וCSS
  • name - שם השדה כפי שיישלח לשרת
  • value - הערך ההתחלתי או הערך שיישלח לשרת
  • placeholder - טקסט שיוצג בשדה כהנחיה למשתמש ונעלם כשמתחילים להקליד
  • required - מסמן שהשדה הוא שדה חובה
  • disabled - מבטל את האפשרות לערוך את השדה
  • readonly - הופך את השדה לקריאה בלבד (לא ניתן לערוך)

טפסים ב-ASP.NET:

ב-ASP.NET Web Forms, אנחנו משתמשים בפקדי שרת (Server Controls) במקום אלמנטי HTML רגילים. הם נראים דומה, אבל יש להם תחילית asp: והם כוללים את המאפיין runat="server". היתרון בשימוש בפקדים אלה הוא שאפשר לגשת אליהם ישירות מקוד C# שכותבים בקובץ code-behind.

פקדי ASP.NET נפוצים לטפסים:

פקד ASP.NET מקביל ב-HTML תיאור דוגמה
TextBox input type="text" שדה טקסט <asp:TextBox ID="txtName" runat="server"></asp:TextBox>
DropDownList select רשימה נפתחת <asp:DropDownList ID="ddlCity" runat="server">
  <asp:ListItem Text="בחר עיר" Value="" />
  <asp:ListItem Text="תל אביב" Value="tlv" />
</asp:DropDownList>
RadioButton input type="radio" כפתור רדיו <asp:RadioButton ID="rbMale" runat="server" GroupName="Gender" Text="זכר" />
CheckBox input type="checkbox" תיבת סימון <asp:CheckBox ID="chkAgree" runat="server" Text="אני מסכים" />
Button input type="button" כפתור <asp:Button ID="btnSubmit" runat="server" Text="שלח" OnClick="btnSubmit_Click" />
Label span תווית להצגת טקסט <asp:Label ID="lblResult" runat="server" Text=""></asp:Label>

דוגמה לשימוש בפקדי ASP.NET בטופס:

                        <div>
                        <asp:Label ID="lblName" runat="server" Text="שם מלא:"></asp:Label>
                        <asp:TextBox ID="txtName" runat="server"></asp:TextBox>
                        </div>

                        <div>
                        <asp:Button ID="btnSubmit" runat="server" Text="שלח" OnClick="btnSubmit_Click" />
                        </div>

המאפיין OnClick="btnSubmit_Click" מקשר את הכפתור לפונקציה בשם btnSubmit_Click בקובץ ה-Code Behind (קובץ ה-C#) שתרוץ כאשר לוחצים על הכפתור.

צעד 10: יצירת דף "צור קשר" בסיסי

                        namespace MyFirstWebsite

                        if (!IsPostBack)

                        protected void btnSubmit_Click(object sender, EventArgs e)

צעד 11: הבנת מאפיינים מיוחדים בפקדי ASP.NET

פקדי ASP.NET מספקים פונקציונליות מתקדמת יותר מאשר תגיות HTML רגילות. הנה כמה מאפיינים מיוחדים:

מאפייני TextBox:

  • TextMode - מגדיר את סוג התיבה: SingleLine (ברירת מחדל), MultiLine (מספר שורות), או Password (להסתרת הקלט)
  • MaxLength - המספר המקסימלי של תווים שאפשר להזין
  • Rows - מספר השורות (למצב MultiLine)
  • Columns - מספר העמודות (למצב MultiLine)
  • ReadOnly - האם השדה לקריאה בלבד

צעד 11: הבנת מאפיינים מיוחדים בפקדי ASP.NET

מאפייני DropDownList:

  • AutoPostBack - אם מוגדר כ-true, הדף יישלח לשרת אוטומטית כאשר משנים את הבחירה
  • SelectedIndex - האינדקס של הפריט הנבחר
  • SelectedValue - הערך של הפריט הנבחר
  • SelectedItem - אובייקט המייצג את הפריט הנבחר
  • Items - אוסף של כל הפריטים ברשימה

מאפייני Button:

  • CommandName - שם הפקודה (משמש כאשר יש מספר כפתורים שמפעילים את אותה פונקציה)
  • CommandArgument - פרמטר שניתן להעביר לפונקציה
  • CausesValidation - האם הכפתור יגרום לבדיקת תקינות (validation) של הטופס בעת לחיצה
  • PostBackUrl - כתובת URL שהדף יועבר אליה בעת לחיצה

מאפייני Label:

  • Text - הטקסט שיוצג
  • ForeColor - צבע הטקסט
  • BackColor - צבע הרקע
  • Font - מאפשר לשנות את הגופן, הגודל, וכו'

הערה חשובה: פקדי ASP.NET רבים תומכים במאפיינים דומים כמו Enabled (האם הפקד פעיל), Visible (האם הפקד מוצג), CssClass (מחלקת CSS), ועוד.

צעד 13: אימות טפסים עם JavaScript

אחד השימושים הנפוצים ביותר של JavaScript הוא אימות (validation) של טפסים בצד הלקוח. זה מאפשר לבדוק את הנתונים לפני שליחתם לשרת ולתת משוב מיידי למשתמש:

יצירת קובץ לבדיקת תקינות:

  1. לחצו קליק ימני על תיקיית Scripts
  2. בחרו Add > New Item
  3. בחרו JavaScript File ותנו לו את השם "validation.js"
  4. לחצו Add

הוסיפו את הקוד הבא לקובץ:

                            // פונקציה ראשית לבדיקת תקינות הטופס
                            function validateForm() {
                            // איפוס הודעות שגיאה קודמות
                            clearErrors();

                            var isValid = true;

                            // בדיקת שם
                            if (!validateName()) {
                            isValid = false;
                            }

                            // בדיקת אימייל
                            if (!validateEmail()) {
                            isValid = false;
                            }

                            // בדיקת נושא
                            if (!validateSubject()) {
                            isValid = false;
                            }

                            // החזרת התוצאה - האם הטופס תקין
                            return isValid;
                            }

                            // בדיקת שם
                            function validateName() {
                            var nameField = document.getElementById("txtName");
                            if (!nameField) return true;  // אם השדה לא קיים, המשך

                            var name = nameField.value.trim();
                            if (name === "") {
                            showError(nameField, "אנא הזן שם");
                            return false;
                            }

                            return true;
                            }

                            // בדיקת אימייל
                            function validateEmail() {
                            var emailField = document.getElementById("txtEmail");
                            if (!emailField) return true;

                            var email = emailField.value.trim();
                            if (email === "") {
                            showError(emailField, "אנא הזן כתובת אימייל");
                            return false;
                            }

                            // בדיקת תבנית אימייל באמצעות ביטוי רגולרי
                            var emailPattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
                            if (!emailPattern.test(email)) {
                            showError(emailField, "אנא הזן כתובת אימייל תקינה");
                            return false;
                            }

                            return true;
                            }

                            // בדיקת נושא
                            function validateSubject() {
                            var subjectField = document.getElementById("ddlSubject");
                            if (!subjectField) return true;

                            if (subjectField.value === "") {
                            showError(subjectField, "אנא בחר נושא");
                            return false;
                            }

                            return true;
                            }

                            // הצגת הודעת שגיאה
                            function showError(field, message) {
                            // יצירת אלמנט להודעת שגיאה
                            var errorElement = document.createElement("div");
                            errorElement.className = "error-message";
                            errorElement.style.color = "red";
                            errorElement.style.fontSize = "smaller";
                            errorElement.innerHTML = message;

                            // הוספה אחרי השדה
                            field.parentNode.insertBefore(errorElement, field.nextSibling);

                            // סימון השדה עם מסגרת אדומה
                            field.style.borderColor = "red";
                            }

                            // מחיקת כל הודעות השגיאה
                            function clearErrors() {
                            // מחיקת הודעות שגיאה קיימות
                            var errorMessages = document.getElementsByClassName("error-message");
                            while (errorMessages.length > 0) {
                            errorMessages[0].parentNode.removeChild(errorMessages[0]);
                            }

                            // איפוס סגנון השדות
                            var formElements = document.getElementsByTagName("input");
                            for (var i = 0; i < formElements.length; i++) {
                            formElements[i].style.borderColor = "";
                            }

                            var selectElements = document.getElementsByTagName("select");
                            for (var i = 0; i < selectElements.length; i++) {
                            selectElements[i].style.borderColor = "";
                            }
                            }
