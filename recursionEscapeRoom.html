Ackermann.java
public static int ackermann(int m, int n) {
    if (m == 0) return n + 1;
    if (n == 0) return ackermann(m - 1, 1);
    return ackermann(m - 1, ackermann(m, n - 1));
}

// Warning: This grows VERY fast!
// ackermann(3, 3) = 61
üí° A(3,3) = 61. This is a famous result in computer science!
