SECTION 01

סטטיסטיקה בסיסית

מדדי מרכז – איפה ה"אמצע"?
center.py
grades = [72,85,90,65,78,92,88,75,80,95, 70,82,87,73,91,68,84,79,86,77] print(f'Min: {min(grades)}') print(f'Max: {max(grades)}') print(f'Mean: {np.mean(grades):.1f}') print(f'Median: {np.median(grades)}')
▼
OUTPUT ▾ Min: 65 ← הנמוך ביותר Max: 95 ← הגבוה ביותר Mean: 80.9 ← סכום / כמות = 1617/20 Median: 81.0 ← הערך האמצעי (אחרי מיון)
📊

Mean (ממוצע)

סכום / כמות. רגיש לחריגים!

📍

Median (חציון)

הערך האמצעי. עמיד לחריגים!

🏷️

Mode (שכיח)

הערך שמופיע הכי הרבה

💡

כשיש חריג (למשל ציון 20): הממוצע צונח ← החציון עדיף! לכן בנתוני שכר תמיד מציגים חציון.

מדדי פיזור ורבעונים Spread

כמה הנתונים "מפוזרים" סביב המרכז?
spread.py
print(f'Range: {np.max(grades) - np.min(grades)}') print(f'Std: {np.std(grades):.2f}') print(f'Q1: {np.percentile(grades, 25)}') print(f'Q2: {np.percentile(grades, 50)}') print(f'Q3: {np.percentile(grades, 75)}') print(f'IQR: {np.percentile(grades,75)-np.percentile(grades,25)}')
▼
OUTPUT ▾ Range: 30 ← 95 - 65 Std: 8.06 ← סטיית תקן (פיזור סביב הממוצע) Q1: 74.25 ← 25% מתחת לערך הזה Q2: 81.0 ← 50% = החציון Q3: 88.25 ← 75% מתחת לערך הזה IQR: 14.0 ← Q3 - Q1 (טווח בין-רבעוני)

📏 סטיית תקן (Std)

Std קטן = ציונים מרוכזים סביב הממוצע
Std גדול = ציונים מפוזרים רחוק מהממוצע
שתי כיתות עם ממוצע 80 יכולות להיות שונות לחלוטין!

📦 IQR (טווח בין-רבעוני)

Q3 - Q1 = הטווח שבו נמצאים 50% האמצעיים
חשוב לזיהוי חריגים:
חריג = מתחת Q1-1.5×IQR או מעל Q3+1.5×IQR

SECTION 02

מהו EDA?

Exploratory Data Analysis – חקר נתונים לפני בניית מודלים

🔑 הכלל הזהב: "Garbage In, Garbage Out"

אם הנתונים לא טובים – גם המודל לא יהיה טוב. EDA = להכיר את הנתונים לעומק לפני שמתחילים!

📊 סטטיסטיקות בסיסיות → 📈 ויזואליזציה → 🔗 קשרים בין משתנים → 🧹 ניקוי והכנה

🏠 הדאטהסט שלנו: דירות למכירה

200 דירות עם: גודל (מ"ר), חדרים, גיל בניין, עיר (ת"א/י-ם/חיפה/ב"ש), מצב (חדש/משופץ/ישן), קומה, מחיר
המטרה: להבין מה משפיע על מחיר הדירה כדי לבנות מודל חיזוי.

גרפי משתנה בודד

Univariate – הבנת כל משתנה בנפרד

📊 היסטוגרמה (Histogram) מספרי

מראה התפלגות – איך הערכים מתפזרים?
histogram.py
plt.hist(df['Price'] / 1e6, bins=20, color='#e74c3c', edgecolor='white') plt.xlabel('Price (Millions ₪)') plt.ylabel('Count') # אפשר גם עם Seaborn: sns.histplot(df['Size_sqm'], bins=20, kde=True)
מה רואים בהיסטוגרמת מחיר? ▾ 📊 רוב הדירות: 1.0-2.5 מיליון ₪ 📈 התפלגות מוטה ימינה (Right Skewed) = יש כמה דירות יקרות מאוד שמושכות את הממוצע 📌 הממוצע (2.1M) > החציון (1.9M) ← סימן להטיה!
kde=True מוסיף קו חלק ▾ 📈 KDE (Kernel Density Estimate) = היסטוגרמה "מוחלקת" שימושי להשוואת התפלגויות בין קבוצות: sns.kdeplot(data=df, x='Price_M', hue='City')
📌

מתי? למשתנים מספריים (גיל, מחיר, גודל). מה לומדים? צורת התפלגות, טווח ערכים, שכיחות.

📦 Box Plot (תרשים קופסה) 5 סיכומים + חריגים

מראה Min, Q1, Median, Q3, Max + נקודות חריגות
boxplot.py
# Box Plot – השוואת מחיר לפי עיר sns.boxplot(data=df, x='City', y='Price_M', hue='City')
מה רואים בגרף? ▾ 📊 תל אביב – הקופסה הכי גבוהה → המחירים הגבוהים ביותר 📊 באר שבע – הקופסה הכי נמוכה → הזולה ביותר ◆ נקודות מעל השפם = חריגים (דירות יקרות במיוחד) 📏 קופסה רחבה = פיזור גדול, צרה = פיזור קטן
ANATOMY OF A BOX PLOT
  ◆ outlier
  ┬ Max (or upper whisker)
  │
┌─┤─┐ Q3 (75%)
│   │
│ ─ │ Median (Q2, 50%)
│   │
└─┤─┘ Q1 (25%)
  │
  ┴ Min (or lower whisker)
  ◆ outlier

SECTION 04

גרפי רב-משתנים

קשרים בין משתנים – החלק הכי חשוב!

📍 Scatter Plot (פיזור) שני משתנים מספריים

scatter.py
plt.scatter(df['Size_sqm'], df['Price_M'], alpha=0.5) plt.xlabel('Size (sqm)') plt.ylabel('Price (Millions ₪)')
▼
מה רואים? ▾ 📈 גודל ↔ מחיר: קשר חיובי חזק! כל עוד הנקודות עולות "ימינה-למעלה" = חיובי 📊 חדרים ↔ מחיר: קשר חיובי מתון (יותר מפוזר כי חדרים = ערכים בדידים) 📉 גיל בניין ↔ מחיר: קשר שלילי חלש (ישן = קצת זול, אבל לא חד-משמעי)
🔑

Scatter Plot הוא הגרף הכי שימושי ב-EDA! מראה קשר, כיוון (חיובי/שלילי), עוצמה, וחריגים – הכל בגרף אחד.

🎨 Scatter + hue (משתנה שלישי) צבע = קבוצות

צובעים את הנקודות לפי משתנה נוסף – רואים אם הקשר שונה בין קבוצות!
scatter_hue.py
sns.scatterplot(data=df, x='Size_sqm', y='Price_M', hue='City', # צבע = עיר size='Rooms', # גודל = חדרים style='Condition', # צורה = מצב alpha=0.7)
▼
עכשיו רואים 5 משתנים בבת אחת! ▾ 📍 X = גודל (מ"ר) 📍 Y = מחיר (מיליון ₪) 🎨 צבע = עיר → ת"א (כתום) מעל כולם! ⚫ גודל נקודה = חדרים → נקודות גדולות = יותר חדרים ◆▲● צורה = מצב → Excellent, Good, Fair 💡 תובנה: לאותו גודל, דירות בת"א יקרות ב-30-50% מב"ש!
📌

hue = צבע לפי קטגוריה. size = גודל לפי מספר. style = צורת נקודה. אפשר לשלב הכל ב-sns.scatterplot!

🎯 זיהוי חריגים (Outliers) IQR Method

ערכים קיצוניים שיכולים להשפיע על המודל
outliers.py
def find_outliers(data, col): Q1 = data[col].quantile(0.25) Q3 = data[col].quantile(0.75) IQR = Q3 - Q1 lower = Q1 - 1.5 * IQR upper = Q3 + 1.5 * IQR outliers = data[(data[col] < lower) | (data[col] > upper)] return outliers, lower, upper out, lo, hi = find_outliers(df, 'Price_M') print(f'Bounds: [{lo:.2f}, {hi:.2f}]') print(f'Outliers: {len(out)}')
▼
OUTPUT ▾ Bounds: [-0.40, 4.40] ← מתחת / מעל = חריג Outliers: 7 ← 7 דירות עם מחיר חריג 💡 מה לעשות? ✂️ הסרה – כשחריגים הם שגיאות 📏 גבול – np.clip(lower, upper) 🔍 חקירה – לפעמים חריגים מעניינים!
