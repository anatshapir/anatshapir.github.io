import pandas as pd data = { 'name': ['Alice', 'Bob', 'Charlie', 'Dana', 'Eve', 'Frank'], 'class': ['A', 'B', 'A', 'B', 'A', 'B'], 'grade': [85, 92, 78, 95, 88, None], 'age': [17, 18, 17, 18, 17, 18], 'gender': ['F', 'M', 'M', 'F', 'F', 'M'] } df = pd.DataFrame(data) print(df)

גישה לעמודות Series vs DataFrame

ההבדל: סוגריים בודדים [ ] = Series, סוגריים כפולים [[ ]] = DataFrame

עמודה אחת → Series

python
df['name']
OUTPUT – Series ▾ 0 Alice 1 Bob 2 Charlie 3 Dana 4 Eve 5 Frank Name: name, dtype: object

מספר עמודות → DataFrame

python
df[['name', 'grade']]
OUTPUT – DataFrame ▾ name grade 0 Alice 85.0 1 Bob 92.0 2 Charlie 78.0 3 Dana 95.0 4 Eve 88.0 5 Frank NaN
⚠️

df['name'] = Series (סוגריים בודדים)  |  df[['name','grade']] = DataFrame (סוגריים כפולים)

loc ו-iloc חשוב מאוד!

גישה לשורות ועמודות ספציפיות

loc – לפי שם (label)

loc.py
df.loc[0:2, 'name':'grade']
⚠️ כולל את שורה 2! ▾ name class grade 0 Alice A 85.0 1 Bob B 92.0 2 Charlie A 78.0

iloc – לפי מיקום (index)

iloc.py
df.iloc[0:2, 0:3]
⚠️ לא כולל שורה 2! ▾ name class grade 0 Alice A 85.0 1 Bob B 92.0
single_row.py
df.loc[3] # שורה בודדת → Series
▼
OUTPUT ▾ name Dana class B grade 95.0 age 18 gender F Name: 3, dtype: object
🎯

loc = Label → כולל 2 קצוות  |  iloc = integer → לא כולל קצה עליון (כמו range)

תנאים מרובים & | ~

שילוב מספר תנאים עם אופרטורים לוגיים
and_filter.py
# AND – ציון מעל 80 וגם כיתה A result = df[(df['grade'] > 80) & (df['class'] == 'A')] print(result)
▼
OUTPUT – grade>80 AND class=='A' ▾ name class grade age gender 0 Alice A 85.0 17 F 4 Eve A 88.0 17 F
or_filter.py
# OR – ציון מעל 90 או גיל 17 result = df[(df['grade'] > 90) | (df['age'] == 17)] print(result)
▼
OUTPUT – grade>90 OR age==17 ▾ name class grade age gender 0 Alice A 85.0 17 F 1 Bob B 92.0 18 M 2 Charlie A 78.0 17 M 3 Dana B 95.0 18 F 4 Eve A 88.0 17 F

💡

query() – נוח כשיש הרבה תנאים, בלי df[...] וסוגריים מיותרים!

dropna ו-fillna טיפול בחסרים

✏️ fillna – מילוי

fillna.py
df['grade'].fillna( df['grade'].mean() )
NaN → ממוצע (87.6) ▾ 0 85.0 1 92.0 2 78.0 3 95.0 4 88.0 5 87.6 ← היה NaN! Name: grade, dtype: float64
options.py
df.dropna(how='all') # רק אם כל השורה NaN df.dropna(subset=['grade']) # לפי עמודה ספציפית df['grade'].fillna(0) # מילוי ב-0 df['grade'].fillna(df['grade'].median()) # מילוי בחציון
⚠️

מחזירות DataFrame חדש! חובה לשמור: df = df.dropna()
